Setup

Headless

  1. download lates .img

  2. delete partitions on microsd

  3. flash .img to microsd with rufus

  4. add empty, extensionless file ssh

  5. add wpa_supplicant.conf and update network:

    country=US ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev update_config=1

    network={ scan_ssid=1 ssid="your_wifi_ssid" psk="your_wifi_password" }

  6. In terminal, ssh pi@ipi.pip.ip.ip

  7. login, password is 'raspberry'

  8. sudo apt update & upgrade

  9. sudo apt install xrdp - rdp won't be able to login with pi, another user must be created

  10. add user USER for xrdp

  11. login with windows rdp
